# DevPath

DevPath is a static HTML, CSS, and JavaScript learning dashboard for developers.

## Pages

- Home: `index.html`
- Login: `pages/login.html`
- Dashboard: `pages/dashboard.html`
- Roadmaps: `pages/roadmap.html`
- Courses: `pages/courses.html`

## Run Locally

Open `index.html` in a browser, or run a local static server:

```powershell
npx serve .
```

Then open the URL shown in the terminal, usually `http://localhost:3000`.

## GitHub Pages

This project can be hosted from the `main` branch using GitHub Pages.
